# ICSE-Class-10-Coding-Projects

Welcome to my ICSE Class 10 coding project repository!

This repository contains beginner-friendly Java programs created as part of my school-level coding practice. The main aim of this project is to improve my Java basics, understand logic-building, and slowly become better at writing clean and correct programs.

## About This Repository

This project includes Java programs based on Class 10 level topics.
The programs are simple, easy to understand, and made for learning purposes.

Basically, this is the place where I write code, fix errors, learn from mistakes, and politely hope the program runs on the first try.

## Topics Covered

Some of the topics included in this repository are:

* Basic Java programs
* If-else statements
* Loops
* Switch-case
* Method overloading
* Area and volume programs
* Series-based programs
* Simple mathematical programs
* Logic-building practice

## Purpose of This Project

The purpose of this repository is to:

* Practice Java programming
* Improve problem-solving skills
* Understand Class 10 coding concepts
* Keep my school coding work organized
* Track my progress as I learn

## Note

This repository is made by a Class 10 ICSE student, so the programs are beginner-level and may improve over time.
Mistakes are also part of learning, but debugging is always ready to teach a lesson.
